# Student Performance Prediction

## Project Overview
This is a machine learning project that predicts student marks based on:
- Study Hours
- Attendance
- Sleep Hours

## Problem Statement
The goal of this project is to predict a student’s academic performance using daily habits and attendance data.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- VS Code

## Machine Learning Model
- Linear Regression

## Workflow
1. Load dataset using pandas  
2. Select input features and output  
3. Split data into training and testing sets  
4. Train model using Linear Regression  
5. Evaluate model accuracy  
6. Visualize data using graphs  

## Input Features
- Study Hours
- Attendance
- Sleep Hours

## Output
- Predicted Marks

## Accuracy
- Model Accuracy: 97%
